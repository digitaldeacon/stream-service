# stream-service

To get started

1. Create a DB
2. clone the project
3. run php init
4. run composer install
5. Take care to use 127.0.0.1 as dbhost (in common/config/main-local.php)
6. Create stream/backend/runtime/sessions
7. run yii migrate in order to create the migration and user table
8. Create a user (if not existant): service/stream/site/signup
